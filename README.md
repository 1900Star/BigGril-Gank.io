PhotoView
============

##所有的图片，都来自 www.gank.io
#一个用ViewPager实现的图片浏览器，图片资源通过Okhttp下载，支持图片放大、缩小、拖动、将图片保存到本地相册
****
## 第三方库
    *buttbutterknife
    *retrofit
    *fresco
    *glide
    *eventbus
